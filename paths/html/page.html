Leaf's Job Portal API

Introduction

This API allows you to integrate your website with our services seamlessly.

Key Features

Easy Integration

Integrate our API with your applications effortlessly.

Secure Access

All requests are secured with industry-standard protocols.

Real-time Data

Get up-to-date data from our services.

API Endpoints

  • GET /jobs

    Retrieves a list of available jobs.

    Example Request/Response 
    POST /jobs
Authorization: Bearer YOUR_API_KEY

Response:

{
  "jobs": [
    {
      "id": 1,
      "title": "Software Engineer",
      "company": "TechCorp",
      "location": "San Francisco, CA"
    },
    ...
  ]
}
  • POST /users

    Displays Information about the user if they applied to one of your applications

    Example Request/Response 
    POST /users
Authorization: Bearer YOUR_API_KEY
Content-Type: application/json

{
  "jobId": 1,
  "applicantName": "John Doe",
  "email": "john@example.com",
}

Response:
{
    "resume": "https://example.com/resume.pdf",
    "applicationId": 12345,
    "name": "John Doe",
    "email": "john@example.com",
    "phone": "1234567890",
    "profilePicture": "https://example.com/profile.jpg"
}
  • GET /applications

    Retrieves jobs posted by an employer (requires employer authentication).

    Example Request/Response 
    GET /applications
Authorization: Bearer YOUR_EMPLOYER_API_KEY

Response:

{
  "jobs": [
    {
      "id": 1,
      "title": "Senior Developer",
      "applicants": 15,
      "status": "active"
    },
    ...
  ]
}
