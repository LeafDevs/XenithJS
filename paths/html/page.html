Leaf's Job Portal API

Introduction

This API allows you to integrate your website with our services seamlessly.

Key Features

Easy Integration

Integrate our API with your applications effortlessly.

Secure Access

All requests are secured with industry-standard protocols.

Real-time Data

Get up-to-date data from our services.

API Endpoints

  • POST /register

    This endpoint allows users to register by providing their email and password. The data is encrypted for security.

  • GET /

    This is the main entry point of the API, providing an overview and access to the API documentation.

  • POST /auth

    This endpoint is used for user authentication, requiring email and password to obtain a session token.

  • GET /jobs

    This endpoint allows users to view available jobs.

  • GET /jobs/:id

    This endpoint allows users to view a specific job by ID.

  • POST /users/:user

    Displays the user's information excluding their password and personal information.

  • GET /users/:user/jobs

    Displays the jobs that the user has applied to.

  • GET /users/:user/jobs/:job

    Displays the job application status of the user.

  • POST /applications

    Displays all the applications.
